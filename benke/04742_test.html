1.1 2.5

您答错了
  • a
    趋向正无穷
  • b
    小
  • c
    趋向1
  • d
    大
参考信息量的定义I=-Log2p
1.2 2.5
功率信号的自相关函数定义为R(τ)，下列说法正确的是（）
您答错了
  • a
    R(τ)=-R(-τ)
  • b
    R(τ)<=>P(f)
  • c
    信号的平均功率P=R(∞)
  • d
    R(τ)反映的是信号的频域特性
R(τ)是偶函数；根据维纳辛钦定理功率信号的自相关函数与功率谱密度是一对傅里叶变换；R(∞)对应着直流功率;R(τ)反映同一信号的不同时刻，时间间隔τ（不同时发生），信号之间的相关性程度。
1.3 2.5
哪一条不是高斯白噪声的特点（）
您答错了
  • a
    噪声均值为0
  • b
    单边功率谱密度为n 0
  • c
    色谱为白色
  • d
    噪声的取值为高斯分布
高斯白噪声中，白的含义不是色谱是白色，而是因为在整个频谱上噪声的双边功率谱密度是一个常数n0/2
